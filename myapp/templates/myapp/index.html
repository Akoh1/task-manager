This is web app to manage tasks for technical teams, You can open a workspace for your organization as an admin and users can sign up to that workspace, create tasks, assign it milestones, change tasks milestones. Admins can coordinate workflow as well as other functionalities for their members while users can only coordinate workflow for themselves. This is a personal project, so it is constantly being improved upon, link to API documentation using swagger will soon be provided This web app also provides RESTFUL API which would be consumed by my front-end project which would be implemented by vuejs later on.
